# Send statement to Databricks

Send statement to Databricks

## Usage

``` r
# S4 method for class 'DatabricksConnection,character'
dbSendStatement(conn, statement, ...)
```

## Arguments

- conn:

  A DatabricksConnection object

- statement:

  SQL statement

- ...:

  Additional arguments (ignored)

## Value

A DatabricksResult object
