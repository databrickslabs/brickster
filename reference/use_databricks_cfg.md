# Returns whether or not to use a `.databrickscfg` file

Returns whether or not to use a `.databrickscfg` file

## Usage

``` r
use_databricks_cfg()
```

## Value

boolean

## Details

Indicates `.databrickscfg` should be used instead of environment
variables when either the `use_databrickscfg` option is set or Posit
Workbench managed OAuth credentials are detected.
