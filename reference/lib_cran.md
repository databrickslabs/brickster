# Cran Library (R)

Cran Library (R)

## Usage

``` r
lib_cran(package, repo = NULL)
```

## Arguments

- package:

  The name of the CRAN package to install.

- repo:

  The repository where the package can be found. If not specified, the
  default CRAN repo is used.
