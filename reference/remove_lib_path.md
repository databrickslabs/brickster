# Remove Library Path

Remove Library Path

## Usage

``` r
remove_lib_path(path, version = FALSE)
```

## Arguments

- path:

  Directory to remove from
  [`.libPaths()`](https://rdrr.io/r/base/libPaths.html).

- version:

  If `TRUE` will add the R version string to the end of `path` before
  removal.
