# Generate Request JSON

Generate Request JSON

## Usage

``` r
db_request_json(req)
```

## Arguments

- req:

  a httr2 request, ideally from
  [`db_request()`](https://databrickslabs.github.io/brickster/reference/db_request.md).

## Value

JSON string
