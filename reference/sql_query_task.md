# SQL Query Task

SQL Query Task

## Usage

``` r
sql_query_task(query_id, warehouse_id, parameters = NULL)
```

## Arguments

- query_id:

  The canonical identifier of the SQL query.

- warehouse_id:

  The canonical identifier of the SQL warehouse.

- parameters:

  Named list of paramters to be used for each run of this job.
