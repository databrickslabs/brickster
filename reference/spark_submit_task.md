# Spark Submit Task

Spark Submit Task

## Usage

``` r
spark_submit_task(parameters)
```

## Arguments

- parameters:

  List. Command-line parameters passed to spark submit. Use Task
  parameter variables to set parameters containing information about job
  runs.
