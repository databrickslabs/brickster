# Close Databricks Workspace Connection

Close Databricks Workspace Connection

## Usage

``` r
close_workspace(host = db_host())
```

## Arguments

- host:

  Databricks workspace URL, defaults to calling
  [`db_host()`](https://databrickslabs.github.io/brickster/reference/db_host.md).

## Examples

``` r
if (FALSE) { # \dontrun{
close_workspace(host = db_host())
} # }
```
